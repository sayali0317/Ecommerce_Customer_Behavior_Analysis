# E-commerce Customer Behavior Analysis (EDA + Visualization)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on an E-commerce Customer Behavior dataset to identify purchasing trends, customer engagement patterns, spending behavior, and satisfaction levels.

The analysis converts raw customer data into business insights through cleaning, preprocessing, statistical exploration, and visualization techniques.

## Objective

* Analyze customer purchasing behavior.
* Identify high-value customer groups.
* Study customer satisfaction and retention indicators.
* Evaluate discount impact on spending.
* Generate business recommendations.

## Dataset Information

Dataset Columns:
* Customer ID
* Gender
* Age
* City
* Membership Type
* Total Spend
* Items Purchased
* Average Rating
* Discount Applied
* Days Since Last Purchase
* Satisfaction Level

### 4. Data Visualization
Created:

* Discount Impact

## Key Business Insights
* Premium membership users generated higher spending.
* Customer satisfaction remained positive across segments.
* Discount campaigns influenced purchasing decisions.
* Certain cities contributed significantly to revenue.
* Customer purchase behavior varied across demographics.
